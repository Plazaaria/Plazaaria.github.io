que es el wordpress

¿Qué es y para qué sirve el WordPress? Qué es WordPress, para qué sirve y cómo utilizarlo WordPress es una plataforma de gestión de contenidos, representa un software de código abierto altamente versátil que permite a los usuarios crear y desarrollar sitios web personalizados de manera sencilla, adaptable y profesional, con una muy buena experiencia de usuario.

WordPress llegó para democratizar la web, como otros CMS. Desde el año 2003, es un sistema de gestión de contenidos que hace que la creación de contenido web no dependa sólo de programadores y de personas de alto conocimiento técnico. Ahora, cualquier persona puede crear una web.
