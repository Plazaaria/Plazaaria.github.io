que es el gif en html

¿Qué es un GIF en HTML? La palabra GIF es el acrónimo de Graphics Interchange Format, lo que se traduce al español como Formato de Intercambio de Gráficos. Se trata de un formato muy utilizado en la web, de forma que se puede afirmar que los GIFs en HTML son elementos que permiten añadir tanto animaciones como imágenes a tu sitio web.

¿Cómo insertar un GIF en HTML? gif html5 En la práctica, los GIFs en HTML no son más que archivos con extensión .gif que se suben al servidor web y se enlazan desde el documento HTML. Para ello, se utiliza la etiqueta , la cual sirve para añadir imágenes a la web.

Se trata de una etiqueta vacía (no tiene etiqueta de cierre) y su particularidad es que siempre debe llevar consigo el atributo src, en cuyo valor se indica la ruta al archivo GIF que deseas mostrar en HTML.
